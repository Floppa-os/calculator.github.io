# Calculator
## Инструкция по использованию калькулятора


### 1. Основные элементы интерфейса


- **Экран отображения** (верхняя часть) — показывает текущее число или результат вычислений.
- **Цифровые кнопки** (0–9) — ввод чисел.
- **Точка (.)** — ввод десятичной дроби.
- **Операторы** (+, −, ×, ÷) — выбор математического действия.
- **Кнопка «C»** (красный цвет) — полная очистка экрана.
- **Кнопка «=»** (зелёный цвет) — вычисление результата.


### 2. Порядок работы

**Шаг 1. Ввод первого числа**  
Нажмите цифровые кнопки для набора первого числа. Например:  
- Для числа *123* нажмите: `1` → `2` → `3`.
- Для дроби *5.7* нажмите: `5` → `.` → `7`.

**Шаг 2. Выбор операции**  
Нажмите кнопку оператора:  
- `+` — сложение,  
- `−` — вычитание,  
- `×` — умножение,  
- `÷` — деление.

**Шаг 3. Ввод второго числа**  
Аналогично шагу 1 введите второе число.

**Шаг 4. Получение результата**  
Нажмите `=` — на экране появится итог вычисления.


### 3. Примеры вычислений


1. **Сложение**:  
   `5` → `+` → `3` → `=` → результат: **8**.

2. **Вычитание**:  
   `10` → `−` → `4` → `=` → результат: **6**.

3. **Умножение**:  
   `6` → `×` → `7` → `=` → результат: **42**.

4. **Деление**:  
   `15` → `÷` → `3` → `=` → результат: **5**.

5. **Дробные числа**:  
   `2.5` → `+` → `1.3` → `=` → результат: **3.8**.

### 4. Особые случаи

- **Очистка экрана**:  
  Нажмите `C`, чтобы сбросить все введённые данные и начать заново.

- **Повторные операции**:  
  После получения результата можно продолжить вычисления, не нажимая `C`. Например:  
  `2` → `+` → `3` → `=` (результат: **5**) → `+` → `4` → `=` (итог: **9**).

- **Деление на ноль**:  
  При попытке деления на ноль калькулятор может выдать ошибку или значение `Infinity` (зависит от реализации).

### 5. Ограничения

- Калькулятор поддерживает **только базовые операции** (+, −, ×, ÷).
- Нет поддержки скобок или сложных выражений (например, `(2+3)*4`).
- Ввод некорректных последовательностей (например, `++5`) может привести к ошибке.

### 6. Советы по использованию

- Для быстрого ввода десятичных чисел используйте точку (`.`).
- Если ошиблись при вводе, нажмите `C` и начните заново.
- Результат автоматически отображается после нажатия `=`.
